# PAHOABC Aggregated Population Data by ADM1

A dataset containing aggregated population information by the first
geographic administrative level (ADM1). Each row provides the population
of children aged 0 and 1 years for a given year and administrative
region.

## Usage

``` r
pahoabc.pop.ADM1
```

## Format

A data frame with X rows and 4 variables:

- ADM1:

  The name of the first geographic administrative level.

- year:

  The year of the population data.

- age:

  The age (in years) of the population for that year and ADM1.

- population:

  The population for that year and ADM1.
