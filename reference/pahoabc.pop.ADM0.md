# PAHOABC Aggregated Population Data by ADM0

A dataset containing aggregated population information for the whole
country (ADM0). Each row provides the population of children aged 0 and
1 years for a given year.

## Usage

``` r
pahoabc.pop.ADM0
```

## Format

A data frame with X rows and 3 variables:

- year:

  The year of the population data.

- age:

  The age (in years) of the population for that year and ADM1.

- population:

  The population for that year and ADM1.
